The info_table is a per-snp information table providing information from 
the VCF record, as well as the daf and gerp scores for each snp. Much of 
this table is taken whole-sale from the INFO field of the VCF record. In
fact, most of the field descriptions come from the phase I integrated 
call sets file header (also included at the bottom).

NOTE THAT THE INFORMATION HERIN IS ONLY PERTINENT TO THE PHASE I INTEGRATED CALL 
SETS! ANY OTHRER VCF SOURCE FILE WILL NOT BE PROCESSABLE BY InfoTableBuilder!
AND THEREFORE THIS TABLE WILL NOT BE GENERATED.

Below are the 30 fields generated by InfoTableBuilder: 

01   Integer chrom 
02   Integer position 
03   String  id 
04   String  filter 
05   Integer qual 
06   Float   LDAF      // MLE Allele Frequency Accounting for LD
07   Float   AVGPOST   // Average posterior probability from MaCH/Thunder   
08   Float   RSQ       // Genotype imputation quality from MaCH/Thunder
09   Float   ERATE     // Per-marker Mutation rate from MaCH/Thunder
10   Float   THETA     // Per-marker Transition rate from MaCH/Thunder
11   String  CIEND     // Confidence interval around END for imprecise variants
12   String  CIPOS     // Confidence interval around POS for imprecise variants
13   Integer END       // End position of the variant described in this record  
14   Integer HOMLEN    // Length of base pair identical micro-homology at event breakpoints
15   String  HOMSEQ    // Sequence of base pair identical micro-homology at event breakpoints
16   Integer SVLEN     // Difference in length between REF and ALT alleles
17   String  SVTYPE    // Type of structural variant
18   Integer AC        // Alternate Allele Count 
19   Integer AN        // Total Allele Count
20   String  AA        // Ancestral Allele, ftp://ftp.1000genomes.ebi.ac.uk/vol1/ftp/pilot_data/technical/reference/ancestral_alignments/README
21   Float   AF        // Global Allele Frequency based on AC/AN
22   Float   AMR_AF    // Allele Frequency for samples from AMR based on AC/AN
23   Float   ASN_AF    // Allele Frequency for samples from ASN based on AC/AN
24   Float   AFR_AF    // Allele Frequency for samples from AFR based on AC/AN
25   Float   EUR_AF    // Allele Frequency for samples from EUR based on AC/AN
26   String  VT        // indicates what type of variant the line represents
27   String  SNPSOURCE // indicates if a snp was called when analysing the low coverage or exome alignment data
28   Double  DAF	   // The derived allele frequency (DAF = DAF_count / (1092*2))
29   int     DAF Count // the number of instances that a derived allele occurs
30   int     TotalTokens  the number of genotypes reported. It should (almost) always equal 1092*2
31	 int InvalidRecords   really these are (apparently) records where a persons genotype has been 
	 overridden by the imputation engine. That is, the genotype predicted by the alignment is not 
	 what the VCF file indicates.
